WinCache Cleaner
Version: v1.0.0 Scope: Windows caches only Safe: skips missing tools

What gets cleaned

  • Android Studio / Gradle caches
  • Flutter projects (build/.dart_tool/.packages/pubspec.lock) + Pub/FVM cache
  • Node package managers (npm / yarn / pnpm)
  • Python pip cache
  • .NET / NuGet cache
  • IDEs: Visual Studio, VS Code, JetBrains
  • Browsers: Chrome, Edge, Brave, Firefox, Opera/Opera GX, Vivaldi
  • Adobe: Premiere, After Effects, Photoshop, Media Encoder, Lightroom
  • Unity / Unity Hub caches
  • Docker Desktop logs/cache (prune optional)
  • System temp and Recycle Bin
No source files or documents are touched.
