WinCache Cleaner

PowerShell utility to clear Windows dev and creative caches (Android Studio/Gradle, Flutter, Node, Python, .NET/NuGet, IDEs, browsers, Adobe, Unity, Docker, temp).

GitHub Stars

Quick run (one-liner)

iwr https://raw.githubusercontent.com/chetanvaghela457/wincache-cleaner/main/wincache-cleaner.ps1 -OutFile wincache-cleaner.ps1; powershell -ExecutionPolicy Bypass -File .\wincache-cleaner.ps1

From a clone

git clone https://github.com/chetanvaghela457/wincache-cleaner.git cd wincache-cleaner powershell -ExecutionPolicy Bypass -File .\wincache-cleaner.ps1

What it cleans

  • Android Studio / Gradle
  • Flutter projects + global cache
  • Node (npm/yarn/pnpm)
  • Python pip cache
  • .NET / NuGet cache
  • IDEs: Visual Studio, VS Code, JetBrains
  • Browsers: Chrome, Edge, Brave, Firefox, Opera/Opera GX, Vivaldi
  • Adobe: Premiere, After Effects, Photoshop, Media Encoder, Lightroom
  • Unity / Unity Hub
  • Docker Desktop logs/cache (optional prune)
  • System temp and Recycle Bin
